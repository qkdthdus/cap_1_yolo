📌 Project Overview

CHA-RIM(차림) 프로젝트는
한국 전통 미학과 기술 기반 인터랙션을 결합한 하이엔드 티 브랜드 경험을 설계하는 것을 목표로 한다.

조선 백자의 절제미, 사방신의 서사, 오행의 개념을 현대적으로 재해석하여
브랜딩 · 패키지 디자인 · 인터랙티브 전시가 유기적으로 연결된 새로운 브랜드 경험을 제안한다.

🎯 2.1.2 프로젝트 목표

한국적 미학 기반 브랜드 아이덴티티(BI) 구축

조선 백자의 순백과 여백의 미

사방신의 상징적 서사를 패키지 디자인에 간접 반영

기술 기반 인터랙티브 브랜드 경험 제공

YOLO 기반 모션 인식 기술 활용

관람객이 브랜드 세계관에 직접 참여하는 전시형 인터랙션 구현

브랜딩 · 광고 비주얼 · 인터랙션의 통합

시각적 디자인에 머무르지 않고

실제 경험 가능한 브랜드 구조 설계

🧭 2.2 브랜딩 전략 수립
2.2.1 브랜드 철학 & 콘셉트

브랜드명 : CHA-RIM (차림)

‘차를 차리다’와 ‘마음을 차리다’라는 중의적 의미를 담아,
차 한 잔을 통해 스스로를 정돈하고 평온한 상태로 나아가는 준비 과정을 상징한다.

차를 단순한 음료가 아닌
삶의 리듬을 조율하는 매개체로 인식하게 하는 브랜드 철학

🎨 디자인 철학

사방신·오행을 직접적으로 드러내지 않음

조선의 순백 질감, 여백의 미를 핵심 디자인 언어로 설정

절제된 형태를 통해 방향성과 시간 개념을 은유적으로 표현

또한 평면 시안에 그치지 않고,
실제 촬영된 패키지 이미지에 디자인을 적용하여
현실감과 브랜드 완성도를 강화하였다.

2.2.2 차(Tea) 종류 선정 & 사방신·오행 해석

사방신과 오행을 정서적 웰빙 기준으로 재해석하여 차를 구성하였다.
한의학의 장부 개념을 참고하되, 기능성 제품이 아닌
스스로의 상태를 인식하고 선택하는 경험에 초점을 둔다.

사방신	대응 장부	차 종류
청룡(木)	간	녹차
주작(火)	심장	결명자차
백호(金)	폐	도라지차
현무(水)	신장	검은콩차

➡️ 맛과 향이 아닌 컨디션과 감정에 따라 선택하는 차라는 브랜드 스토리 형성

2.2.3 디자인 모티프 & 오행 시각화

오행의 성질을 추상적 도형 언어로 변환하여
설명 없이도 각 사방신의 성격이 직관적으로 전달되도록 설계하였다.

목(木) : 성장 · 상승
→ 수직적이고 곧은 직선

화(火) : 확산 · 열정
→ 에너지 분출을 표현한 삼각형

금(金) : 결실 · 응집
→ 안정감 있는 원형 / 반원

수(水) : 흐름 · 지혜
→ 부드러운 곡선과 물결

이 도형 모티프는
패키지 그래픽 → 촬영 연출 → 비주얼 브랜딩 전반으로 확장된다.

2.2.4 촬영 콘셉트 & 시간의 해석

사방신이 관장하는 방향과 시간을 핵심 메타포로 삼아
총 4가지 라인업을 구성하였다.

방향	사방신	시간대	시각적 요소
동(木)	청룡	아침	직선 · 청록색 · 대나무
남(火)	주작	낮	삼각형 · 붉은 산 능선
서(金)	백호	저녁	원형 · 백색 & 금색
북(水)	현무	밤	물결선 · 흑색 계열

➡️ 하루의 흐름과 감정 변화를 차로 경험하도록 설계

2.2.5 패키지 구조 & 재질(Materiality)

6면 입체 패키지 구조

각 면을 개별 디자인 후 촬영 이미지에 합성

다양한 시안을 실험 가능

재질 표현

도자기(세라믹)의 매트한 질감

한지 특유의 미세한 엠보싱 구현

한국적 고급스러움 강조

Cut-out 기법 적용

측면·후면까지 이어지는 형태 흐름

내부 오행 그래픽 노출

금박·은박 후가공 효과로 시각적 깊이감 확보
